# Mercimek Space

## Description (WIP)

- A hackerspace in Ankara.
- The hackerspace no one asked for, but definitely needed.
- The hackerspace Ankara didn't ask for, but definitely needed.

## Philosophy/manifesto (WIP)

For a long time, we have been building things.
We coded, painted, soldered, broke stuff, fixed it, broke it again, and called it progress.
And we have done all of this from apartments, cafes, labs, and random corners of Ankara.
It's time to get together in this chaos.

Mercimek: A space for hackers, makers, artists, musicians, outsiders and the quietly curious.

P.S. We don't know what we are doing. Maybe that's the point.
